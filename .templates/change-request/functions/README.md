# Change Request templates theo chức năng

> Mỗi chức năng chính của repo có 2 template: `*.feature.md` (thêm/ mở rộng) và `*.fix.md` (sửa lỗi). Tất cả kế thừa `../TEMPLATE_change-request.md`. Chọn đúng file → copy làm task → điền → làm → cập nhật ngược.

| Chức năng | Feature | Fix |
|---|---|---|
| Credential theo owner | `credential.feature.md` | `credential.fix.md` |
| Fetch Builder / Flow | `fetch-builder.feature.md` | `fetch-builder.fix.md` |
| Placeholder Engine | `placeholder-engine.feature.md` | `placeholder-engine.fix.md` |
| Variables + Extracted Data | `variables.feature.md` | `variables.fix.md` |
| History & Logs | `history-logs.feature.md` | `history-logs.fix.md` |
| Inspect Mode | `inspect-mode.feature.md` | `inspect-mode.fix.md` |
| Issues CRUD | `issues.feature.md` | `issues.fix.md` |
