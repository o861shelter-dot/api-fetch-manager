## C. QUY TẮC BẤT BIẾN (tóm tắt AGENTS.md — vi phạm = reject)

- **Backend:** env prefix `API_FETCH_MANAGER_` · credential AES-256-GCM at-rest, trả masked (trừ `/reveal` có audit) · log che token · advanced JS sandbox timeout 200ms · giữ interface `Db` · response `{ ok, data?, error? }` · HTTP ngoài timeout+retry.
- **Frontend:** KHÔNG `alert/confirm/prompt` → modal · modal có ✕, click ngoài không đóng, tự scrollbar · button icon+tooltip · màu qua CSS variable · **mọi danh sách dùng `DataList`** · **docs không che UI**.
- **credId:** mỗi credential tự sinh credId; placeholder map theo credId; key nhiều giá trị → KeyPicker buộc chọn; fallback theo key chỉ khi duy nhất 1 giá trị.
- **RTDB:** 5 DB cũ bất biến + #6 `rtdb-resources`; `.indexOn` đúng.
- **Bảo mật:** secret doc gốc đã lộ → rotate; seed/test dữ liệu GIẢ.
