Endpoint Overview

The Authentication endpoint exchange is required for all stateful sessions. Ensure grant_type is set to client_credentials for service-to-service communication.

Error Responses
  • 401: Invalid client credentials
  • 403: Insufficient scope permissions
  • 429: Rate limit exceeded

Extraction Guide

Use JSONPath syntax to extract tokens. Variables stored in the Global context will persist across all builder steps in the current environment.
